# 丸進引越センター ランディングページ

伊丹・西宮・尼崎の地域密着引越し「丸進引越センター」（丸進運輸株式会社）のマーケティング用ランディングページ。

Claude Design でモックアップした `丸進引越センター LP.dc.html` を、依存ライブラリなしの**静的サイト**（HTML / CSS / Vanilla JS）として実装したものです。

## 公開URL（GitHub Pages）

https://aimayujin0911.github.io/marushin-hikkoshi-lp/

## 構成

```
.
├── index.html        # ページ本体（インラインCSS + 末尾に Vanilla JS）
├── assets/
│   ├── logo.png      # ロゴ
│   └── trucks.jpeg   # ヒーロー画像（引越トラック）
└── .nojekyll         # GitHub Pages の Jekyll 処理を無効化
```

## インタラクション

- **30秒かんたん見積もり** — 「人数・荷物量 × 移動距離 × 時期」を選ぶと概算料金（¥low〜¥high）をその場で再計算。
  係数・計算式は元デザインの `DCLogic` クラスをそのまま移植。
- **FAQ アコーディオン** — クリックで開閉。初期表示は先頭の項目を開いた状態。

## ローカル確認

```bash
python -m http.server 8000
# → http://localhost:8000
```
