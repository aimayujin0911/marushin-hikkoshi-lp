FAQ

よくあるご質問

Q 見積もりは無料ですか？ +

はい、お見積もりは無料です。お電話・フォームから承り、概算のご相談だけでもお気軽にどうぞ。

Q 急な引越しでも対応できますか？ +

年中無休（年末年始を除く）で受付。空き状況により当日〜翌日のご相談も承ります。まずはお電話ください。

Q 荷造りや開梱もお願いできますか？ +

荷造りから運搬・開梱・設置までワンストップで対応します。お忙しい方向けのおまかせコース（かるがも）もご用意しています。

Q エアコンやピアノも運べますか？ +

エアコンの電気工事、ピアノ輸送・調律、自動車・ペット輸送など、幅広いオプションに対応しています。

Q 対応エリアはどこまでですか？ +

伊丹・西宮・尼崎を中心に、全国・海外まで対応します。遠方のお引越しもお気軽にご相談ください。

Q 不用品の処分もできますか？ +

お引越しに合わせて不用品の処分やトランクルームのご利用も承ります。お見積もり時にご相談ください。
